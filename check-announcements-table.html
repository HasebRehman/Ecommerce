🔍 Announcements Table Checker

This tool checks if the announcements table exists in your Supabase database.

⚠️ Before Running This Check:

  1. Make sure your .env file has the correct Supabase credentials
  2. Your development server should be running (npm run dev)
  3. You should be logged in as Super Admin or Platform Admin

📝 If Table Doesn't Exist:

  1. Open your Supabase Dashboard: https://supabase.com/dashboard
  2. Go to SQL Editor → New Query
  3. Open the file RUN_THIS_SQL.md in this project
  4. Copy the SQL code and paste it into the Supabase SQL Editor
  5. Click Run
  6. Come back here and click "Check Table Status" again
